# 🍲 StreetConnect

## Digitising India’s Street Food Ecosystem 🇮🇳

---

## 📌 Overview

StreetConnect is a mini prototype built to digitalise the street food sector of Indian markets.  
Street vendors frequently change stall locations based on crowd flow and demand, making it difficult for customers to find them consistently.

This platform creates a digital bridge between vendors and users — allowing real-time location updates, pre-orders, and community-based meetups.

---

## 🚀 Features

### 👨‍🍳 Vendor Features
- 📍 Update daily stall location (Google Maps integration)
- 📝 Update menus anytime
- 🔔 Daily reminder notifications to check and update status
- 🎙️ Gemini voice assistance for guided updates
- 🛍️ Accept pre-orders for quick pickup
- 💳 Two-part payment system:
  - Partial online payment (pre-order)
  - Remaining payment in person

---

### 👥 User Features
- 🔍 Discover nearby street food vendors
- 📌 View updated locations & menus
- 🛒 Pre-order food to save time
- 👨‍👩‍👧‍👦 Form groups with users visiting the same stall
- 💬 In-app temporary group chat
- ⭐ Subscribe to favorite vendors for live updates

---

## 🔐 Privacy & Community

- Group chats are accessible only inside the app
- No personal details are shared
- Chats dissolve automatically after meetup
- Encourages real-world interaction 🤝
- Promotes community building over screen dependency
